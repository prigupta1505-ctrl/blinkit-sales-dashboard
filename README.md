# 🛒 Blinkit Sales Dashboard (Power BI)

## 📊 Project Overview
This project is a Power BI dashboard created to analyze Blinkit sales data and generate meaningful business insights.

## 🔑 Key Insights
- 💰 Total Sales: 393K
- 📦 Total Items: 2785
- 📈 Average Sales: 141
- ⭐ Average Rating: 3.9

## 📌 Features
- Sales performance analysis
- Category-wise insights
- Interactive filters
- Visual storytelling using charts

## 🛠 Tools Used
- Power BI
- Data Visualization
